# ScamShield V1 🛡️

ScamShield is an explainable AI system that analyzes message text to detect scam risk and explain **why** a message may be dangerous.

Unlike traditional spam filters, ScamShield combines:
- Machine learning–based risk estimation
- Rule-based scam tactic detection
- Human-readable explanations

---

## 🚩 Features (V1)

- Scam risk score (0–100)
- Risk levels: Low / Medium / High
- Detection of scam tactics:
  - OTP solicitation
  - Urgency pressure
  - Authority impersonation
  - Fear or threat-based coercion
  - Financial pressure
- Explainable decision output
- API-ready backend
